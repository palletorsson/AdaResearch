# Swarm Intelligence Algorithms

Collective behavior systems inspired by natural swarms and social organisms.

## Algorithms Included

- **Ant Colony Optimization** (`ant_colony_optimization/`) - Path finding through pheromone trails.
- **Boids Algorithm** (`boids_algorithm/`) - Flocking behavior from simple local rules.
- **Particle Swarm Optimization** (`particle_swarm_optimization/`) - Optimization via cooperative particles.
